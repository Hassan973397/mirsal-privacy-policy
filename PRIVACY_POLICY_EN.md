## 2. Information We Collect

### 2.1 Information You Provide Directly

When you register for Mirsal app, we collect the following information:

- **Account Information**: Email address, full name, phone number
- **Merchant Information** (for merchants): Business name, business type, address, business license number
- **Driver Information** (for drivers): Vehicle type, license plate number, license number
- **Location Information**: Geographic location for delivery tracking (collected only when using tracking feature)

### 2.2 Information We Collect Automatically

- **Device Information**: Device type, operating system, unique device identifier
- **Usage Information**: How you use the app, features you use, usage time
- **Network Information**: IP address, connection type (Wi-Fi/cellular data)
- **Location Information**: Precise geographic location when using delivery tracking feature (collected only when tracking is enabled)

### 2.3 Order-Related Information

- **Order Information**: Order details, delivery addresses, customer information
- **Payment Information**: Payment and billing information (if available)
- **Delivery Information**: Delivery status, delivery proof (photos), delivery timestamps

## 7. Required Permissions

Mirsal app requires the following permissions:

### 7.1 Camera
- **Purpose**: Scanning barcodes and capturing delivery proof
- **Usage**: Only when using barcode scanning or delivery proof features

### 7.2 Location
- **Purpose**: Tracking delivery location and proving delivery location
- **Usage**: Only when delivery tracking feature is enabled
- **Access**: Only while using the app (Location When In Use)

### 7.3 Notifications
- **Purpose**: Sending notifications about orders and updates
- **Usage**: To inform you about order status and important updates

### 7.4 Internet
- **Purpose**: Connecting to servers to sync data
- **Usage**: Essential for app functionality

### 7.5 Photos (Photo Picker)
- **Purpose**: Selecting photos for delivery proof or banners
- **Usage**: We use Photo Picker API which does not require direct photo access permissions

**Note**: We do not request direct photo access permissions (READ_MEDIA_IMAGES/VIDEO) in accordance with Google Play policy. We use Photo Picker API which provides a secure interface for selecting photos.
